# Data.md

The data/modeling material has been split:

- `FPGA.md` covers running DIMACS files on an F2 instance and collecting hardware-side CSVs.
- `Model.md` covers turning measured CSVs into scaling projections with the existing scripts.
- `Scaling.md` remains the theory/reference document behind the model.

If you were looking for the old combined data-and-modeling guide, start with `Model.md`.
